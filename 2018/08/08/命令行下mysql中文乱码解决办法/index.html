命令行下mysql中文乱码解决办法

2018-08-08
cmd, mysql
阅读量次

前言

cmd命令行下执行sql遇到中文乱码的问题，如下图查询语句

问题原因

cmd客户端默认编码为”gbk”编码，mysql我设置的默认编码是”utf8”,在cmd中指明mysql的编码和想要的编码即可。

操作步骤

  1. 打开cmd,输入mysql -u root -p，输入密码进入mysql交互界面
  2. 设置编码set character_set_client=gbk;
  3. 再设置set character_set_results=gbk;
    再重新执行查询语句看到中文已经显示正常
