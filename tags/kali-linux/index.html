Kali Linux使用SSH远程登陆连接

新安装的 Kali Linux 不能直接使用SSH工具来远程登陆，有些参数需要做以下修改。 修改 sshd_config 文件中的参数命令：vim /etc/ssh/sshd_config将 PermitRootLogin 的注释解除，并将值设置为 yes 将 PasswordAuthentication 的注释解除，并将值设置为 yes 保存并退出vim 开启SSH服务命令：service ss
